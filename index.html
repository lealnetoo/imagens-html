Imagens no HTML

No início a Web era somente texto, e era tedioso. Felizmente, não demorou muito para que a capacidade de incorporar imagens (e outros tipos de conteúdo mais interessantes) dentro das páginas da web fosse adicionada. Existem outros tipo de mídia para se considerar, mas é lógico começar com o humilde elemento <img>, usado para inserir uma simples imagem em uma página web. Neste artigo, analisaremos como dominar seu uso, incluindo o básico, anotando-o com legendas usando o elemento <figure>.

Como colocamos uma imagem numa página web?

Para colocar uma imagem numa página web, você precisa usar o elemento <img>. Este é um elemento vazio, o que significa que ele não tem um elemento de fechamento, e você não deve colocar nada entre o elemento de abertura e fechamento. Ele requer apenas um atributo, src, que informa ao navegador onde ele pode encontrar a imagem que você deseja exibir. O valor do atributo src é o URL da imagem que você deseja exibir. Por exemplo, se você tiver uma imagem chamada meu-gato.jpg em um diretório chamado imagens em seu site, você poderia usar o seguinte código HTML para exibir a imagem:

Por exemplo, se sua imagem for chamada dinossauro.jpg, e está no mesmo diretório de sua página HTML, você poderia inserir a imagem assim:

<img src="dinossauro.jpg" />

Se a imagem estiver em um diretório diferente, você precisará especificar o caminho para a imagem em relação ao arquivo HTML. Por exemplo, se a imagem estiver em um diretório chamado imagens que está no mesmo diretório que o arquivo HTML, você poderia usar o seguinte código HTML:

<img src="imagens/dinossauro.jpg" />

E assim por diante.

Você pode incorporar a imagem usando seu URL absoluto, por exemplo:

<img src="https://www.example.com/imagens/dinossauro.jpg" />

Texto alternativo

É uma boa prática adicionar um atributo alt ao seu elemento <img>. O texto alternativo é o texto que é exibido se a imagem não puder ser exibida. Isso pode ser porque o arquivo da imagem está faltando, porque o URL é digitado incorretamente, ou porque o usuário está usando um leitor de tela para navegar na página. O texto alternativo é importante para ajudar os usuários com deficiência visual a entender o conteúdo da imagem.

Por exemplo, se você tiver uma imagem de um dinossauro, você poderia adicionar o texto alternativo "Dinossauro" assim:

<img src="dinossauro.jpg" alt="Dinossauro" />

Largura e altura

Se você quiser especificar a largura e a altura da imagem, você pode fazer isso usando os atributos width e height. Por exemplo, se você quiser exibir uma imagem de 200 pixels de largura e 100 pixels de altura, você poderia usar o seguinte código HTML:

<img src="dinossauro.jpg" alt="Dinossauro" width="200" height="100" />

É uma boa prática especificar a largura e a altura da imagem, pois isso permite que o navegador reserve espaço para a imagem enquanto ela é carregada, evitando que o conteúdo da página salte para cima e para baixo enquanto a imagem é carregada.

Títulos de imagem

Se você quiser adicionar um título à sua imagem, você pode fazer isso usando o atributo title. O título é exibido quando o usuário passa o mouse sobre a imagem. Por exemplo, se você quiser adicionar um título "Dinossauro" à sua imagem, você poderia usar o seguinte código HTML:

<img src="dinossauro.jpg" alt="Dinossauro" title="Dinossauro" />

É uma boa prática adicionar títulos às suas imagens, pois isso ajuda os usuários a entender o conteúdo da imagem e o contexto em que ela está sendo usada.

Anotar imagens com figuras e legendas de figuras

Se você quiser adicionar uma legenda à sua imagem, você pode fazer isso usando o elemento <figure>. O elemento <figure> é usado para envolver uma imagem e sua legenda, e o elemento <figcaption> é usado para adicionar a legenda. Por exemplo, se você quiser adicionar uma legenda "Dinossauro" à sua imagem, você poderia usar o seguinte código HTML:

<figure>
    <img src="dinossauro.jpg" alt="Dinossauro" />
    <figcaption>Dinossauro</figcaption>
</figure>

É uma boa prática adicionar legendas às suas imagens, pois isso ajuda os usuários a entender o conteúdo da imagem e o contexto em que ela está sendo usada.

Conclusão

Adicionar imagens a uma página web é uma maneira poderosa de tornar o conteúdo mais interessante e envolvente. O elemento <img> é a maneira mais simples de adicionar uma imagem a uma página web, e você pode personalizar a aparência e o comportamento da imagem usando os atributos src, alt, width, height e title. Você também pode adicionar legendas às suas imagens usando o elemento <figure> e o elemento <figcaption>. Esperamos que este artigo tenha ajudado você a entender como adicionar imagens a uma página web e a usar as melhores práticas para tornar suas imagens acessíveis e envolventes.
